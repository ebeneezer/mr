## 2026-04-15 - Add keyboard shortcut to About Dialog 'Done' button
**Learning:** In the Turbo Vision framework used by this app, keyboard shortcuts for dialog buttons (which allow users to press Alt+letter) are defined by simply enclosing the shortcut letter in tildes in the button text (e.g., '~D~one'). This is a common pattern for adding keyboard accessibility.
**Action:** When adding new dialog buttons, ensure a keyboard shortcut is defined using the tilde syntax to maintain keyboard accessibility.
