# Multi-Edit Revisited (mr) 

- is a rewrite of the classic programmers editor Multi-Edit by American Cybernetics for Linux terminals.
- MEMAC level (now called MRMAC) is version 3.x of Multi-Edit and the code uses TVision from magiblot on github for the UI.
- mr uses piecetables to edit fiels larger than systemstorage, and a lexx/yacc/bison optimized stackmachine for the mrmac language.
- The mrmac language 
	- is capable of steering Tvision 
	- will be used to expand mr in the future 
	- will implement Multi-Edit standard features 
		- cursor movement
		- block management
		- keystroke macros ...


> [!NOTE]
> I am looking for devs knowing Multi-Edit and his fantastic features that are into retro coding. Lets recreate a true terminal gem!
